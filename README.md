# flask and react deployed on heroku


# Usage
* Local Development
`git clone...`
`cd <>`
* Server
`virtualenv -p python3 venv`
`pip install -r requirements.txt`
`gunicorn app:app`
* Client
`cd client-dev`
`npm install`
`npm start`
To build client app: `npm run build`
